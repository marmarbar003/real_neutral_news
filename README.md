# Real Neutral News
## By: Lauren Crawford () & Maria Martin-Barbera (10955852)

## Problem Statement
In this day and age media can be used for anything. Therefore, this project aims to determine whether an article is real or fake. In addition, media can be used to influence individuals' perspective, hence it also detects whether the article is right leaning or left learning. This issue has been addressed using python.

## How to run this code?
- First, please make a virtual environment
- Then run the following command to ensure you have all the neccesary libraries installed
  `pip install -r requirements.txt`
